# ビルドエラー修正コマンド

TypeScriptとビルドエラーを段階的に修正します。

## 実行手順

1. **ビルドを実行**: `npm run build` または `pnpm build`

2. **エラー出力を解析**:
   - ファイルごとにグループ化
   - 重要度でソート

3. **各エラーについて**:
   - エラーのコンテキストを表示（前後5行）
   - 問題を説明
   - 修正案を提案
   - 修正を適用
   - ビルドを再実行
   - エラーが解決されたことを確認

4. **停止条件**:
   - 修正が新しいエラーを導入した場合
   - 同じエラーが3回試行後も残る場合
   - ユーザーが一時停止を要求した場合

5. **サマリーを表示**:
   - 修正されたエラー
   - 残っているエラー
   - 新しく導入されたエラー

## 重要な原則

**安全性のため、一度に1つのエラーを修正してください！**

## 使用タイミング

- TypeScriptのコンパイルエラーが発生した時
- ビルドプロセスが失敗した時
- 型エラーを修正する必要がある時
